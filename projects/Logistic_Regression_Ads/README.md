# Logistic Regression Classification

## Overview
This project implements Logistic Regression to solve binary classification problems. It covers training the models, evaluating accuracy, and serializing the trained models so they can be deployed in web applications later.

## Contents
* `Logisitc_Regression.ipynb` & `Cancer_Dataset_Model.ipynb`: Notebooks for predicting Social Network Ads purchases and cancer classification.
* `Social_Network_Ads.csv`: The dataset used for ad prediction.
* **Saved Models:** Serialized `.pkl` and `.joblib` files containing the exported models ready for deployment.
