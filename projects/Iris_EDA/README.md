# Exploratory Data Analysis: Iris Dataset

## Overview
This folder focuses on Exploratory Data Analysis (EDA). Before applying complex machine learning algorithms, it is crucial to understand the data. This project walks through visualizing and analyzing the feature distributions of the Iris dataset.

## Contents
* `EDA_Iris.ipynb`: Notebook containing statistical summaries and visual distributions of the data.
* `iris.csv`: The dataset containing Iris flower measurements.
