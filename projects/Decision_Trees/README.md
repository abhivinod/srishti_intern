# Decision Tree Classification

## Overview
This project explores the Decision Tree algorithm for classification tasks. It applies tree-based logic to different datasets to predict categorical outcomes based on feature splits.

## Contents
* `decision_tree.ipynb`: Notebook applying Decision Trees to the Social Network Ads dataset.
* `Iris_Tree.ipynb`: Notebook demonstrating Decision Tree classification on the classic Iris dataset.
* `Social_Network_Ads.csv`: Dataset containing user demographics and ad purchasing behavior.
