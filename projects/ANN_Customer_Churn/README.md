# Customer Churn Prediction (ANN)

## Overview
This project focuses on building an Artificial Neural Network (ANN) to predict bank customer churn. By analyzing demographic and account details, the model identifies patterns that indicate whether a customer is likely to leave the bank.

## Contents
* `Ann.ipynb`: Jupyter notebook containing data preprocessing, model building with TensorFlow/Keras, and performance evaluation.
* `Churn_Modelling.csv`: The dataset used for training and testing the network.
