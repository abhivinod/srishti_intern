# Machine Learning Projects

This directory contains the core machine learning projects developed during the internship. Each subfolder represents a specific algorithm or technique applied to a dataset.

## Project Breakdown

* **`Iris_EDA/`** * **Focus:** Exploratory Data Analysis (EDA)
  * **Description:** Initial data exploration and visualization using the classic Iris dataset to understand feature distributions before modeling.

* **`Logistic_Regression_Ads/`** * **Focus:** Binary Classification
  * **Description:** Applied Logistic Regression to predict outcomes for Social Network Ads and Cancer datasets. Includes saved model files (`.pkl`, `.joblib`) for deployment testing.

* **`Decision_Trees/`** * **Focus:** Tree-based Classification
  * **Description:** Implemented Decision Tree classifiers to analyze the Iris dataset and Social Network Ads.

* **`KMeans_Clustering/`** * **Focus:** Unsupervised Learning
  * **Description:** Used the K-Means algorithm to group retail data (`Mall_Customers.csv`) based on customer spending scores and income.

* **`ANN_Customer_Churn/`** * **Focus:** Deep Learning / Artificial Neural Networks
  * **Description:** Built a basic ANN to predict bank customer churn based on account and demographic details (`Churn_Modelling.csv`).
