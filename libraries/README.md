# Libraries Practice

This folder contains my foundational practice and experiments with core Python data science libraries. Getting comfortable with these tools was the first step in my machine learning workflow.

## Contents
* **NumPy (`numpy.ipynb`, `array.npy`):** Practice with array manipulations, numerical operations, and data formatting.
* **Pandas (`pandas.ipynb`, `employee.csv`):** Hands-on work with dataframes, data cleaning, and basic data manipulation.
* **Matplotlib (`matplotlib.ipynb`, `plot.png`):** Basic data visualization, plotting graphs, and formatting visual outputs.

These notebooks serve as a quick reference for data manipulation techniques used in the main projects.
